c++ stl栈stack介绍

C++ Stack（堆栈） 是一个容器类的改编，为程序员提供了堆栈的全部功能，主要操作包括堆栈初始化、元素入栈、取栈顶元素、元素出栈、判断堆栈是否非空和取得当前堆栈大小等。

c++ stl栈stack的头文件为:

#include

c++ stl栈stack的成员函数介绍（5种）

empty() 堆栈为空则返回真

pop() 移除栈顶元素

push(i) 在栈顶增加元素

size() 返回栈中元素数目

top() 返回栈顶元素

栈的底层容器可以是vector、list等，如：

stack<int,vector > s1;
stack<int, list > s2; // 用双向链表作堆栈的底层结构
